# **App Name**: ClinicConnect

## Core Features:

- Specialization Directory: Browse and select medical specializations (e.g., Cardiology, Dermatology) to view available 'doctors'.
- Doctor Listings: Display a static list of fictional doctors for the chosen specialization with basic info like name and experience.
- Fictional Clinic Map: An interactive, fake map displaying the simulated locations of 'clinics' or 'doctor's offices'.
- Doctor Profile View: Clickable doctor profiles showing static detailed information, 'availability', and 'patient reviews'.
- AI-Powered Specialization Overview Tool: Generates a brief, helpful overview or common symptom tips related to the currently selected medical specialization.

## Style Guidelines:

- Primary color: A professional and calming medical blue (#4589A9) to instill trust and clarity.
- Background color: A very light, desaturated blue (#EEF5F8) creating an airy and clean aesthetic.
- Accent color: A fresh and inviting cyan-green (#66CCC2), analogous to the primary, used for interactive elements and highlights.
- All text uses 'Inter', a clean, modern sans-serif font known for its excellent readability and neutral appearance, fitting for professional medical information.
- Utilize simple, easily recognizable medical and UI-related icons for clear navigation and categorization.
- Implement a clear, spacious layout with a consistent two-column structure where applicable, making information easy to scan and digest.
- Subtle and swift transition animations for state changes and navigation, enhancing user feedback without distraction.
