# Clinic Hamani - Premier Healthcare Directory

Clinic Hamani is a high-fidelity prototype of a modern medical directory system. It helps patients find and connect with verified medical specialists through an intuitive and AI-enhanced interface.

## 🚀 Key Features

### 1. AI-Powered Medical Insights
Instead of static descriptions, we use **Genkit with Google Gemini** to dynamically generate overviews for every medical specialization.
- **Contextual Knowledge**: Explains complex specializations in plain English.
- **Symptom Tips**: Generates AI-driven wellness tips and common symptoms.
- **On-Demand**: Content is generated specifically for the specialization you are browsing.

### 2. Extensive Staff Directory
- **20+ Specialists**: A pre-populated database of simulated doctors across various fields.
- **Verified Profiles**: Profiles include experience levels, ratings, and peer/patient reviews.
- **Interactive Map**: A visual discovery tool to find doctors by location.

### 3. Smart Marketing Tools
- **LinkedIn Generator**: An AI tool for healthcare professionals to generate promotional LinkedIn copy instantly based on their specialization.

## 🛠 Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI**: Genkit with Google Gemini (1.5 Pro)
- **Styling**: Tailwind CSS & ShadCN UI
- **Icons**: Lucide React
